# Terraform AWS Infrastructure Automation

This project automates AWS infrastructure provisioning using Terraform.
It demonstrates Infrastructure as Code (IaC) principles for cloud resource management.
The project creates an EC2 instance in AWS using reusable Terraform configurations.
A Security Group is configured to allow SSH (22) and HTTP (80) access.
Terraform variables are used to make the deployment flexible and reusable.
AWS credentials are managed through AWS CLI configuration.
The infrastructure can be previewed using `terraform plan`.
Resources are deployed automatically using `terraform apply`.
Terraform state management enables tracking and updating infrastructure changes.
The project showcases practical Terraform and AWS skills for DevOps and Cloud Engineer roles.

## Technologies Used

- Terraform
- AWS EC2
- AWS Security Groups
- AWS IAM
- AWS CLI
- Git & GitHub
